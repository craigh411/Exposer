Class Exposer Test Suite Instructions

If you would like to run the tests you will need to update the phpunit.xml.dist config file bootstrap path to point to your autoloader
and rename to phpunit.xml. You will also need PHP Unit (http://www.phpunit.de) to be installed and correctly configured on your system and will
need to update the location of the phpunit config file using the -c flag from the command ling or from inside your IDE.
